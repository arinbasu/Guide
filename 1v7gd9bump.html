The bibliography file
The bibliography file is kept at bibliography/biblio.bib -- a bibtex file. The bibtex file can be filled in from any source and if you work with markdown, then use Latex citation style to cite the sources. Authorea will adjust the citation and the references.

Step 2: Link with github

Once you have set up the article (or say you use this as a template and this one contains a jupyter file as well), you can just set up the github links and you can work from github or git and jupyter to wwork with the Authorea article. You can adjust the main sections once your data analyses are complete but otherwise you can push your files from within your computer or from the web using the github web interface.
